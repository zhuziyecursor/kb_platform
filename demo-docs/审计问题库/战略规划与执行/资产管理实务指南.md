# 资产管理实务指南

## 第一章 资产管理概述

### 1.1 资产分类

| 资产类别 | 内容 | 管理要点 |
|---------|------|---------|
| 固定资产 | 房屋、建筑物、设备 | 实物管理 |
| 无形资产 | 专利、软件、土地 | 价值管理 |
| 长期投资 | 股权投资、债权投资 | 投资回报 |
| 流动资产 | 存货、应收账款 | 周转效率 |

### 1.2 资产管理目标

- 资产安全完整
- 资产高效使用
- 资产价值保值增值

---

## 第二章 固定资产管理

### 2.1 配置与使用

**配置程序**：需求申报 → 可行性论证 → 预算审批 → 采购实施 → 验收登记

**日常管理**：资产编号管理、使用人责任、维护保养、定期盘点

### 2.2 处置审批权限

| 原值金额 | 审批层级 |
|---------|---------|
| 10万元以下 | 分管领导 |
| 10-50万元 | 总经理 |
| 50万元以上 | 党委会 |

---

## 第三章 无形资产管理

### 3.1 管理范围

- 专利权、商标权
- 著作权
- 土地使用权
- 软件著作权

### 3.2 管理要求

**专利管理**：申请与维护、许可使用、侵权保护

**土地管理**：使用权登记、开发管理、出租出借
