# 财务管理指引

## 第一章 财务管理体系

### 1.1 财务管理目标

- 确保企业资产安全完整
- 提高资金使用效益
- 保证财务报告真实可靠
- 防范财务风险

### 1.2 财务管理原则

| 原则 | 内容 |
|------|------|
| 合法性 | 严格执行财经法规 |
| 真实性 | 如实反映财务状况 |
| 效益性 | 追求最佳经济效益 |
| 安全性 | 保障资金安全 |

---

## 第二章 预算管理

### 2.1 预算编制原则

- 量入为出
- 统筹安排
- 重点保障
- 厉行节约

### 2.2 预算执行控制

**预算控制**：
- 刚性控制：工资、社保等刚性支出
- 弹性控制：运行费用等弹性支出

**预算调整**：
- 季度调整：常规调整
- 年度调整：重大变化调整

---

## 第三章 资金管理

### 3.1 现金管理

**库存现金**：限额管理，超过限额部分当日送存银行

**银行存款**：规范账户开立和撤销，每月核对银行对账单

### 3.2 资金使用审批

| 金额范围 | 审批人 |
|---------|-------|
| 1万元以下 | 部门负责人 |
| 1-10万元 | 分管领导 |
| 10-50万元 | 总经理 |
| 50万元以上 | 党委会 |

---

## 第四章 资产管理

### 4.1 固定资产管理

**管理要求**：归口管理、分级管理、定期盘点

### 4.2 应收账款管理

- 客户信用评估
- 账龄分析
- 分级催收

---

## 第五章 财务分析

### 5.1 主要分析指标

**偿债能力**：资产负债率、流动比率、速动比率

**运营能力**：存货周转率、应收账款周转率

**盈利能力**：毛利率、净利率、净资产收益率
