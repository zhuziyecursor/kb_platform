# 企业级审计规范

## 第一章 企业内部审计

### 1.1 内部审计定位

内部审计是企业内部设置的独立监督机构，客观评价和改进组织的风险管理、内部控制和公司治理效果。

### 1.2 内部审计职能

| 职能 | 内容 |
|------|------|
| 监督职能 | 经济活动合规性检查 |
| 评价职能 | 内部控制有效性评估 |
| 咨询职能 | 管理改进建议提供 |
| 鉴证职能 | 财务信息真实性确认 |

## 第二章 审计范围

### 2.1 财务审计

财务报表审计、财务收支审计、资产审计

### 2.2 业务审计

经营审计、管理审计、效益审计

### 2.3 专项审计

经济责任审计、舞弊审计、信息系统审计

## 第三章 审计程序

### 3.1 审计准备

确定审计项目 → 组织审计力量 → 制定审计方案 → 下达审计通知

### 3.2 审计实施

进驻被审计单位 → 收集审计证据 → 编制审计工作底稿 → 形成审计发现

### 3.3 审计报告

撰写报告 → 征求意见 → 报批报告 → 下达决定
