# 国家级审计规范

## 第一章 国家审计概述

### 1.1 国家审计定义

国家审计是指国家审计机关依法对国家机关、国有企事业单位财政财务收支及相关经济活动的真实性、合法性、效益性进行监督的行为。

### 1.2 国家审计特征

| 特征 | 说明 |
|------|------|
| 法定性 | 依据法律规定开展 |
| 强制性 | 依法享有强制执法权 |
| 独立性 | 独立于被审计单位 |
| 综合性 | 监督范围覆盖全面 |

## 第二章 审计范围与内容

### 2.1 审计范围

- 中央财政收支、地方财政收支
- 国有金融机构、国有企业
- 政府投资项目

### 2.2 审计内容

| 类型 | 主要内容 |
|------|---------|
| 财政审计 | 预算执行、决算编制 |
| 金融审计 | 资产负债、风险管理 |
| 企业审计 | 资产运营、收益分配 |
| 投资审计 | 项目建设、资金使用 |

## 第三章 审计程序

### 3.1 审计计划

年度审计项目计划 → 中期审计规划 → 长期审计规划

### 3.2 审计实施

审计准备 → 审计实施 → 审计报告 → 审计处理
