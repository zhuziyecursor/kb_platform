# 资本运营投资操作指引

## 第一章 总则

### 1.1 资本运营类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 兼并收购 | 收购或合并目标公司 | 规模扩张 |
| 股权转让 | 出售或转让股权 | 资源优化 |
| 资产重组 | 整合内部资产 | 结构优化 |
| 上市融资 | 首次公开募股 | 融资发展 |

---

## 第二章 并购投资

### 2.1 并购程序

**目标筛选** → **尽职调查** → **交易结构** → **交割整合**

### 2.2 尽职调查内容

- 财务尽调
- 法律尽调
- 业务尽调
- 人力资源尽调

---

## 第三章 投资管理

### 3.1 投资额度管理

| 投资类型 | 审批层级 | 额度限制 |
|---------|---------|---------|
| 参股投资 | 总经理 | 单笔≤5000万 |
| 控股投资 | 董事会 | 单笔≤2亿 |
| 重大投资 | 党委会 | 不限 |

### 3.2 投后管理

**治理管理**：派出人员管理、重大事项报告、利益分配

**风险管理**：风险监测预警、风险应对处置、定期风险评估

---

## 第四章 资产处置

### 4.1 处置程序

```
资产清查 → 评估定价 → 方案审批 → 公开处置 → 款项回收
```

### 4.2 产权交易原则

- 原则上必须进产权交易所
- 公开挂牌、竞价交易
- 防止暗箱操作
