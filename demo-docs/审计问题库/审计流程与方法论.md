# 审计流程与方法论

## 第一章 审计方法论基础

### 1.1 风险导向审计理念

现代审计采用风险导向审计方法，以风险评估为起点，将审计资源集中于高风险领域，提高审计效率和效果。

**风险评估三要素**：
- 固有风险：未考虑控制措施前的风险水平
- 控制风险：内部控制未能防止或发现的风险
- 检查风险：审计抽样未能发现重大错报的风险

### 1.2 审计重要性概念

**重要性阈值**：

| 科目 | 重要性水平 |
|------|-----------|
| 净利润 | 税前利润的5%~10% |
| 营业收入 | 营业收入的0.5%~1% |
| 资产总额 | 资产总额的0.5%~1% |

**重要性判断因素**：
- 金额大小
- 性质严重程度
- 是否涉及违规违纪
- 是否影响报表使用者决策

## 第二章 审计计划阶段

### 2.1 项目计划编制

**计划内容**：
1. 项目背景和目的
2. 被审计单位基本情况
3. 审计范围和重点
4. 时间安排和人员分工
5. 审计方法和程序
6. 重要风险领域
7. 质量控制要求

### 2.2 风险评估程序

**了解被审计单位**：
- 所处行业环境
- 业务模式和发展战略
- 组织架构和治理结构
- 经营目标和财务业绩

**了解内部控制**：
- 控制环境
- 风险评估过程
- 控制活动
- 信息与沟通
- 内部监督

**穿行测试**：
- 按业务主线条线记录处理流程
- 识别关键控制点
- 验证控制是否有效运行

## 第三章 审计实施阶段

### 3.1 实质性测试程序

**分析性程序**：
- 比较同期数据变化
- 分析异常波动
- 验证数据逻辑关系
- 预测期望值

**细节测试**：
- 函证：向第三方确认账户余额
- 盘点：实地盘点库存现金和存货
- 检查：查阅原始凭证和文件
- 重新计算：验证计算准确性

### 3.2 函证程序

**函证类型**：

| 类型 | 适用范围 |
|------|---------|
| 积极式函证 | 重大余额、高风险项目 |
| 消极式函证 | 小额余额、低风险项目 |
| 双重函证 | 重要且存疑的项目 |

**函证控制要点**：
- 函证由审计人员直接寄发和收取
- 被审计单位不得接触函证原件
- 对未回函的积极函证应采取替代程序

### 3.3 存货监盘程序

**监盘前准备**：
- 了解存货存放地点
- 制定监盘计划
- 通知被审计单位配合

**监盘执行**：
- 观察盘点过程
- 检查存货状况
- 抽盘部分存货与盘点记录核对
- 记录盘点结果

**特殊存货处理**：
- 异地存货：委托当地机构盘点或发函确认
- 在途存货：检查相关运输单据
- 受托存货：向存货所有人函证

## 第四章 审计抽样技术

### 4.1 统计抽样方法

**属性抽样**：用于测试控制的执行效果
- 估计偏差率
- 确定样本规模
- 评价抽样结果

**变量抽样**：用于推断总体金额
- 估计总体错报
- 确定样本规模
- 评价抽样结果

### 4.2 非统计抽样

**样本设计**：
1. 定义总体和抽样单元
2. 确定样本规模
3. 选择样本方法
4. 评价样本结果

**样本选取方法**：
- 随机选样
- 系统选样
- 随意选样
- 判断选样

### 4.3 影响样本量的因素

| 因素 | 影响 |
|------|------|
| 可接受风险 | 风险越低，样本量越大 |
| 预期误差率 | 预期误差越大，样本量越大 |
| 总体规模 | 总体越大，样本量越大（但非线性关系） |
| 可容忍误差 | 可容忍误差越大，样本量越小 |

## 第五章 审计报告阶段

### 5.1 审计调整事项

**调整原则**：
- 区分重要程度
- 考虑性质因素
- 与管理层沟通
- 取得管理层声明

**审计调整类型**：
1. 科目余额调整
2. 分类调整
3. 列报调整
4. 前期影响调整

### 5.2 审计报告类型

| 类型 | 适用情形 |
|------|---------|
| 标准无保留意见 | 不存在重大问题 |
| 带强调事项段 | 存在重大不确定事项 |
| 保留意见 | 存在重大但不影响整体报表的错报 |
| 否定意见 | 错报影响重大且广泛 |
| 无法表示意见 | 获取证据不足 |

### 5.3 与管理层沟通

**沟通事项**：
- 审计中发现的问题
- 内部控制缺陷
- 持续经营相关事项
- 审计报告类型

**沟通时间**：
- 审计开始：审计范围和时间安排
- 审计中期：重要发现和进展
- 审计结束：最终审计意见

## 第六章 专项审计方法

### 6.1 经济责任审计

**审计重点**：
- 贯彻执行重大决策情况
- 履行党风廉政建设情况
- 重大投资项目管理情况
- 国有资产管理情况
- 财务管理情况

**审计评价方法**：
- 定量评价：对比目标完成情况
- 定性评价：工作成效和管理水平
- 综合评价：优、良、中、差

### 6.2 建设工程审计

**审计内容**：
- 基本建设程序合规性
- 工程招标规范性
- 合同签订和履行情况
- 工程造价真实性
- 资金使用合规性

**审计方法**：
- 查阅项目立项、批复文件
- 核查招标投标资料
- 核对工程量清单
- 验证工程变更签证
- 实地查看工程项目

### 6.3 信息系统审计

**审计范围**：
- 信息系统治理
- 信息系统安全
- 信息系统运行
- 业务连续性

**审计重点**：
- 访问控制有效性
- 数据完整性
- 系统开发和变更
- 业务系统与财务系统一致性

## 附录：常用审计公式

**样本量计算**：
```
n = (Z × σ / E)²
其中：
Z = 置信系数
σ = 标准差
E = 可容忍误差
```

**总体错报推断**：
```
错报金额 = 样本错报金额 × (总体规模 / 样本规模)
```

**穿行测试覆盖率**：
```
覆盖率 = 实际测试的控制点数 / 应该测试的控制点总数 × 100%
```
