# 审计业务基础知识手册

## 第一章 审计概述

### 1.1 审计的定义

审计是由独立的专业机构或人员，依据国家法规和审计标准，运用专门的方法，对被审计单位的财政财务收支及有关经济活动的真实性、合法性、效益性进行审查，评价经济责任，维护财经法纪，改善经营管理，提高经济效益的一种独立性经济监督活动。

### 1.2 审计的目的

1. **真实性**：验证会计报表和记录是否如实反映被审计单位的财务状况
2. **合法性**：检查经济活动是否符合法律法规和规章制度的要求
3. **效益性**：评价资源使用的效率和效果

### 1.3 审计的分类

| 分类标准 | 类型 | 说明 |
|---------|------|------|
| 按主体分类 | 国家审计 | 审计机关依法进行的审计监督 |
| | 内部审计 | 单位内部审计机构进行的审计 |
| | 社会审计 | 注册会计师依法执行的审计 |
| 按内容分类 | 财务审计 | 财务报表的真实性、合法性 |
| | 绩效审计 | 资金使用效益和管理效率 |
| | 合规审计 | 法规遵循情况 |

## 第二章 审计基本流程

### 2.1 审计准备阶段

1. **确定审计项目**：根据年度计划或上级交办确定审计对象
2. **组织审计组**：选派具有专业胜任能力的人员
3. **编制审计方案**：明确审计目标、范围、重点和方法
4. **发出审计通知**：提前3日向被审计单位送达审计通知书

### 2.2 审计实施阶段

1. **进驻被审计单位**：出示审计证件和审计通知书
2. **收集审计证据**：通过检查、监盘、查询、函证等方法
3. **编制审计工作底稿**：记录审计过程和结果
4. **形成审计发现**：对问题进行定性并提出处理建议

### 2.3 审计报告阶段

1. **撰写审计报告**：对审计结果进行总结
2. **征求被审计单位意见**：给予陈述和申辩的机会
3. **出具审计结论**：下达审计决定书
4. **跟踪整改**：督促被审计单位整改落实

## 第三章 内部审计实务

### 3.1 内部审计的范围

- 财务计划及单位预算的执行情况
- 内部控制制度的健全性和有效性
- 重大投资、融资活动的合规性
- 资产管理的安全性和效益性
- 舞弊行为的预防和查处

### 3.2 内部审计常用方法

| 方法 | 适用场景 |
|------|---------|
| 顺查法 | 适用于业务量较小的单位 |
| 逆查法 | 侧重控制测试，提高效率 |
| 抽查法 | 业务量大时采用抽样审计 |
| 详细审计 | 重要领域或高风险区域 |

### 3.3 审计抽样技术

**统计抽样**：根据随机原则从总体中抽取样本，适用于大型数据集。

**非统计抽样**：审计人员凭借职业判断抽样，适用于小型审计或探索性测试。

**样本量确定因素**：
- 总体规模
- 可接受的风险水平
- 预计误差率
- 预期精确度

## 第四章 常见审计问题及案例

### 4.1 财务审计常见问题

1. **收入确认不及时**：应在提供服务的会计期间确认收入，而非收款时
2. **成本费用跨期**：将本期费用推迟到下期，虚增本期利润
3. **资产虚增**：虚列存货或固定资产
4. **负债隐瞒**：不及时确认负债和或有负债

### 4.2 合规审计关注重点

- "三重一大"决策程序是否规范
- 采购招标是否合规
- 费用报销是否真实
- 资产管理是否安全

### 4.3 典型案例：某单位招待费违规问题

**发现问题**：审计组对某单位进行例行审计时发现，2024年3月存在多笔大额招待费支出，合计金额超过预算控制数的3倍。

**审计方法**：
1. 调阅财务账簿，逐笔核对招待费支出
2. 核查发票真伪和消费明细
3. 约谈经办人员和相关领导

**审计结论**：该单位存在超预算列支招待费、虚列招待事项等问题，涉及金额58万元。

**处理建议**：
1. 追回违规款项
2. 相关责任人作出书面检查
3. 完善招待费管理制度

## 第五章 审计沟通与报告

### 5.1 审计沟通技巧

1. **倾听**：充分听取被审计单位的意见
2. **表达**：用清晰、客观的语言陈述审计发现
3. **协调**：在坚持原则的前提下寻求共识
4. **反馈**：及时通报审计进展和重要事项

### 5.2 审计报告撰写要点

```
报告结构：
├── 审计背景
├── 审计范围和内容
├── 审计依据
├── 审计发现
│   ├── 问题描述
│   ├── 违规事实
│   ├── 定性依据
│   └── 处理建议
├── 审计评价
└── 整改要求
```

### 5.3 审计整改闭环管理

1. **整改跟踪**：建立问题整改台账
2. **整改时限**：一般问题30日内整改完成
3. **整改验收**：审计组确认整改效果
4. **整改报告**：被审计单位提交整改报告
