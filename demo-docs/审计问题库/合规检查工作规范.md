# 合规检查工作规范

## 第一章 总则

### 1.1 目的和依据

为规范合规检查工作流程，确保合规检查的规范性、有效性和及时性，依据《中华人民共和国审计法》《企业内部审计基本准则》等法规制度，制定本规范。

### 1.2 适用范围

本规范适用于企业内部各项合规检查工作，包括：
- 年度例行合规检查
- 专项合规检查
- 上级交办的合规检查任务
- 举报线索核查

### 1.3 合规检查的定义

合规检查是指对被检查单位（或部门）执行国家法律法规、企业内部规章制度情况的监督检查活动。

## 第二章 检查分工与职责

### 2.1 检查组织部门职责

| 部门 | 职责 |
|------|------|
| 审计委员会 | 审议年度检查计划，监督检查工作 |
| 监察审计部 | 制定检查方案，组织实施检查 |
| 业务归口部门 | 配合提供相关业务资料 |
| 人力资源部 | 提供人员信息，配合追责 |

### 2.2 检查人员要求

- 熟悉相关法律法规
- 掌握检查业务流程
- 具有良好的职业道德
- 保守工作秘密

---

## 第三章 检查程序

### 3.1 准备阶段

**工作内容**：
1. 收集被检查单位基本情况
2. 了解相关业务流程和制度
3. 分析历史问题和风险点
4. 制定检查方案
5. 下达检查通知

**检查方案内容**：
- 检查目的和依据
- 检查范围和内容
- 检查时间和人员
- 检查方法和程序
- 工作要求

### 3.2 实施阶段

**主要工作方法**：

| 方法 | 作用 | 适用场景 |
|------|------|---------|
| 访谈 | 了解情况、验证问题 | 管理层、业务人员 |
| 查阅 | 提取书面证据 | 制度文件、记录台账 |
| 核对 | 验证数据一致性 | 账账核对、账实核对 |
| 复算 | 验证计算准确性 | 费用计算、指标核算 |
| 盘点 | 确认资产存在性 | 库存现金、存货盘点 |

**工作底稿要求**：
- 载明检查事项、时间、地点、内容
- 记录检查方法和步骤
- 摘录相关证据资料
- 如实反映发现的问题
- 由检查人员和被检查人签字确认

### 3.3 报告阶段

**检查报告内容**：
1. 检查背景
2. 检查范围和内容
3. 检查时间和对象
4. 检查方法
5. 主要问题及事实
6. 问题定性及依据
7. 处理建议
8. 整改要求

**报告出具流程**：
```
起草报告初稿 → 内部复核 → 征求被检查单位意见 → 修改完善 → 报批 → 下达
```

### 3.4 整改阶段

**整改要求**：
- 一般问题15个工作日内整改
- 复杂问题30个工作日内整改
- 涉及制度建设的问题45个工作日内整改

**整改验收**：
- 被检查单位提交整改报告
- 检查组跟踪验证
- 形成整改验收结论

## 第四章 重点检查内容

### 4.1 "三重一大"决策合规

**检查要点**：
- 重大决策是否经集体讨论决定
- 重要人事任免是否履行程序
- 重大项目安排是否合规
- 大额资金使用是否审批

### 4.2 招标采购合规

**检查要点**：
- 是否应招标未招标
- 招标程序是否规范
- 评标是否公正
- 合同签订是否合规

### 4.3 费用管理合规

**检查要点**：
- 费用报销是否真实
- 差旅费是否超标准
- 招待费是否合规
- 补贴发放是否合规

### 4.4 资产管理合规

**检查要点**：
- 资产购置是否合规
- 资产处置是否审批
- 资产出租出借是否规范
- 无形资产使用是否合规

## 第五章 违规问题处理

### 5.1 问题分类

| 类别 | 情形 | 处理方式 |
|------|------|---------|
| 轻微违规 | 初次违规、情节轻微 | 批评教育、书面警告 |
| 一般违规 | 造成一定影响 | 通报批评、扣减绩效 |
| 严重违规 | 造成较大损失 | 纪律处分、经济赔偿 |
| 违法违纪 | 涉嫌违法 | 移送司法机关处理 |

### 5.2 处理程序

1. **调查核实**：听取当事人陈述，收集证据
2. **定性处理**：依据规定提出处理建议
3. **批准执行**：按管理权限报批后执行
4. **申诉受理**：被检查单位有申诉权利
