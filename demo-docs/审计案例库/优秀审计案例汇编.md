# 优秀审计案例汇编

## 前言

为总结审计工作经验，发挥典型案例示范作用，现汇编一批优秀审计案例，供各级审计人员学习参考。

---

## 案例一：某集团招待费违规使用审计

### 基本情况

**被审计单位**：某国有企业集团总部

**审计时间**：2024年第一季度

**审计背景**：根据年度审计计划，对该集团进行例行财务审计

### 审计方法

1. **数据分析筛选**：运用ACL软件对近三年招待费支出进行趋势分析，锁定异常增长区间

2. **发票核查**：通过税务系统对大额招待费发票进行真伪查询

3. **供应商延伸**：对接待量排名前三的酒店进行实地走访和业务核实

4. **个人账户追踪**：对部分大额支出的收款个人账户进行银行交易流水核查

### 审计发现

| 问题类型 | 涉及金额 | 涉及笔数 |
|---------|---------|---------|
| 虚列招待费 | 128万元 | 23笔 |
| 超标准接待 | 56万元 | 15笔 |
| 报销凭证不全 | 34万元 | 8笔 |

### 处理结果

1. 收回违规款项218万元
2. 对3名责任人进行通报批评
3. 责成完善《招待费管理办法》
4. 建议人力资源部扣减相关人员年度绩效

### 经验总结

- 运用数据分析技术提高审计效率
- 善于从异常数据中发现线索
- 注重外围调查取证实

## 案例二：建设工程招标违规审计

### 基本情况

**项目名称**：某市中心医院新建大楼项目

**投资规模**：4.8亿元

**审计时间**：2023年6月-9月

### 审计方法

1. **程序审查**：查阅招标公告、招标文件、评标报告等全流程资料

2. **评标复核**：组织专家对评标过程进行复核，重点关注评分标准执行

3. **现场核实**：对实际施工与合同约定进行比对

4. **价格核查**：对主要材料价格进行市场询价

### 审计发现

**主要问题**：

1. **围标串标**：三家中标单位实际控制人为同一人，涉及标段金额1.2亿元

2. **中标价虚高**：经市场询价，主要材料价格高出市场价15%-25%，虚增造价约2800万元

3. **资质挂靠**：实际施工方与中标方不符，涉及工程款约3600万元

4. **变更不规范**：未经审批的设计变更累计增加造价约900万元

### 处理结果

1. 终止原中标合同，重新组织招标
2. 追回虚高造价款项2800万元
3. 将围标串标线索移送建设主管部门
4. 对相关责任人员进行处理处分

### 经验总结

- 建设工程招标是审计重点领域
- 要关注中标后的实际施工情况
- 充分利用专家资源和市场询价手段

## 案例三：信息系统安全审计

### 基本情况

**被审计单位**：某市政务服务中心

**审计时间**：2024年2月-3月

**审计目的**：评估信息系统安全管理水平

### 审计方法

1. **文档审阅**：查阅信息安全管理制度、应急预案、运维记录

2. **漏洞扫描**：使用专业工具对系统进行漏洞扫描

3. **渗透测试**：在授权范围内进行模拟攻击测试

4. **访谈调研**：对信息部门负责人、运维人员、安全管理员进行访谈

5. **日志分析**：分析系统访问日志和安全日志

### 审计发现

**高风险问题**：

| 问题 | 风险等级 | 描述 |
|------|---------|------|
| 弱口令 | 高 | 存在大量弱密码，包含admin等默认账户 |
| 权限管理混乱 | 高 | 用户权限过大，离职账户未及时清理 |
| 审计日志缺失 | 中 | 部分系统未开启审计功能 |
| 补丁更新滞后 | 中 | 操作系统存在多个高危漏洞未修复 |
| 数据备份不足 | 低 | 备份周期过长，无异地备份 |

### 整改建议

1. 立即修改所有弱口令，实施密码策略
2. 全面清理账户权限，原则按最小权限分配
3. 开启所有关键系统审计日志
4. 制定补丁更新计划，定期更新
5. 完善备份策略，建立异地灾备

### 经验总结

- 信息系统安全是信息化时代审计的新领域
- 需要借助专业技术工具
- 审计人员应持续学习新技术新知识

## 案例四：经济责任审计

### 基本情况

**被审计领导**：某县教育局局长张某

**审计期间**：2020年9月-2023年8月（三年任期）

**审计时间**：2023年9月-10月

### 审计内容

1. **贯彻执行重大决策情况**
   - 检查三重一大事项集体决策情况
   - 核实重大投资项目的合规性

2. **履行党风廉政建设情况**
   - 检查廉洁从业相关规定执行
   - 核查个人有关事项报告

3. **国有资产管理情况**
   - 检查教育经费使用效益
   - 核查固定资产管理

4. **个人经济责任**
   - 任期内财政收支情况
   - 重大经济决策责任

### 审计评价

**定量评价**：

| 指标 | 目标值 | 实际值 | 完成率 |
|------|-------|-------|-------|
| 教育经费投入增长 | ≥5% | 6.8% | 136% |
| 新建学校数量 | 3所 | 4所 | 133% |
| 教师培训完成率 | 100% | 98% | 98% |

**定性评价**：

张某同志在任期内，认真贯彻党的教育方针，落实上级决策部署，推进教育改革发展，在学校基础设施建设、教育质量提升等方面取得较好成效。但在资产管理和工程招标方面存在不规范问题。

### 审计结果

1. 对存在问题责令整改
2. 审计评价：良（存在一般性问题）

### 经验总结

- 经济责任审计要客观全面
- 定量与定性评价相结合
- 注重区分前任责任与现任责任

## 案例五：采购业务舞弊审计

### 基本情况

**被审计单位**：某科技公司采购部

**审计时间**：2024年1月

**审计背景**：收到举报信反映采购人员收受贿赂

### 审计方法

1. **举报线索分析**：详细分析举报信内容，列出关键线索

2. **供应商筛选**：对举报涉及的供应商进行全面背景调查

3. **数据异常分析**：分析近两年采购数据，查找价格异常和供应商集中度问题

4. **银行流水核查**：在获得授权后核查相关人员银行账户

5. **访谈突破**：对关键证人进行访谈取得证言

### 审计发现

**查实问题**：

1. **商业贿赂**：采购经理李某收受供应商回扣共计87万元

2. **利益输送**：通过指定供应商从中收取好处费，涉及采购金额3200万元

3. **关联交易**：采购人员亲属设立公司与本单位进行交易，涉及金额450万元

4. **招标造假**：在招标过程中泄露标底，与供应商串通投标

### 处理结果

1. 将案件移交司法机关处理
2. 解除涉案人员劳动合同
3. 终止涉案供应商合作关系
4. 完善采购内控机制

### 经验总结

- 举报线索要高度重视
- 数据分析是发现舞弊的有效手段
- 注意保护举报人信息安全
