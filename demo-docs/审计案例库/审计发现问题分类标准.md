# 审计发现问题分类标准

## 第一章 总则

### 1.1 目的

为规范审计发现问题分类，统一问题定性标准，提高审计报告质量和整改针对性，特制定本标准。

### 1.2 适用范围

适用于各类审计项目发现问题的分类和定性。

---

## 第二章 问题分类体系

### 2.1 按问题性质分类

| 类别 | 定义 | 示例 |
|------|------|------|
| 违规问题 | 违反法律法规强制性规定 | 私设小金库、截留收入 |
| 违纪问题 | 违反党纪政纪规定 | 超标准配备公车、违规接待 |
| 损失浪费 | 因管理不善造成的损失 | 工程长期停工、设备闲置 |
| 运行缺陷 | 内控设计或执行不到位 | 审批程序缺失、权限过大 |

### 2.2 按问题金额分类

**重大问题**：
- 涉及金额100万元及以上
- 或占相关资金总量的10%及以上

**较大问题**：
- 涉及金额50万元-100万元
- 或占相关资金总量的5%-10%

**一般问题**：
- 涉及金额50万元以下
- 或占相关资金总量的5%以下

### 2.3 按问题原因分类

| 原因类别 | 说明 |
|---------|------|
| 主观故意 | 明知故犯，蓄意违规 |
| 业务不熟 | 因能力不足导致的失误 |
| 制度缺陷 | 内控制度不健全 |
| 执行不力 | 有制度但未有效执行 |

## 第三章 常见问题定性标准

### 3.1 财务类问题

**虚列支出**：
- 定性依据：《会计法》第9条、第13条
- 表现：伪造、变造会计凭证、会计账簿
- 处理：追回虚列资金，给予处分

**隐瞒收入**：
- 定性依据：《会计法》第10条
- 表现：不按规定确认、计量收入
- 处理：补缴税款，收回隐瞒资金

**坐支现金**：
- 定性依据：《现金管理暂行条例》
- 表现：用现金收入直接支付支出
- 处理：责令改正，给予处罚

### 3.2 采购类问题

**应招标未招标**：
- 定性依据：《招标投标法》第3条
- 金额标准：400万元以上工程、100万元以上货物
- 处理：责令改正，处项目合同千分之十罚款

**围标串标**：
- 定性依据：《招标投标法》第32条、第53条
- 表现：投标人相互串通投标
- 处理：中标无效，处中标项目千分之十罚款

**中标价虚高**：
- 定性依据：采购内控制度
- 表现：合同价格明显高于市场价
- 处理：核减合同价格，追回差价

### 3.3 资产类问题

**账实不符**：
- 定性依据：《会计法》第17条
- 差异处理：查明原因，分类处理
- 账龄分析：3年以上存货、应收账款

**资产闲置**：
- 定性依据：资产管理制度
- 时间标准：闲置超过12个月
- 处理：调剂使用或处置

**资产流失**：
- 定性依据：国有资产管理规定
- 表现：未经批准处置资产
- 处理：停止处置，追回流失资产

### 3.4 决策类问题

**三重一大违规**：
- 定性依据："三重一大"决策制度
- 表现：未经集体研究决定重大事项
- 处理：认定决策无效，追究责任

**决策失误**：
- 定性依据：重大决策终身追责制度
- 损失标准：造成50万元以上直接损失
- 处理：区分容错与问责情形

## 第四章 问题定性的影响因素

### 4.1 从重情节

1. 涉及金额巨大
2. 情节恶劣、性质严重
3. 造成重大损失或恶劣影响
4. 屡查屡犯、拒不整改
5. 涉及违法犯罪

### 4.2 从轻情节

1. 主动交代、积极配合
2. 情节轻微、影响较小
3. 问题发现后及时纠正
4. 历史遗留、涉及面广
5. 属于改革创新探索中的失误

### 4.3 定性优先规则

1. 违法行为优先于违规行为
2. 主观故意优先于过失
3. 造成损失优先于未造成损失
4. 持续进行优先于已停止

## 第五章 问题表述规范

### 5.1 问题表述要素

```
标题（定性） + 问题描述（事实） + 定性依据 + 产生后果 + 处理建议
```

### 5.2 表述示例

**示例一：虚列会议费问题**

标题：某部门虚列会议费支出

问题描述：2023年5月，某部门虚列"年度工作座谈会"会议费支出15万元，用于报销业务招待费用。经查阅会议记录，该座谈会实际未举办。

定性依据：《会计法》第十三条规定"任何单位和个人不得伪造、变造会计凭证、会计账簿和其他会计资料，不得隐匿、销毁依法应当保存的会计凭证、会计账簿和其他会计资料"。

产生后果：导致财务数据失真，违规列支招待费用。

处理建议：责成该部门退回虚列资金15万元，对经办人员进行批评教育，完善会议费报销制度。

**示例二：招标程序不规范**

标题：某工程项目招标程序不规范

问题描述：2023年8月，某单位组织信息系统建设项目招标，中标金额480万元。经查，该项目可行性研究未经批复，招标文件未经集体研究，评标委员会组成不符合规定。

定性依据：《招标投标法》及其实施条例关于招标程序的规定；《某集团工程建设项目招标管理办法》第十五条、第二十条规定。

产生后果：招标结果可能不公正，影响项目质量和资金效益。

处理建议：责成暂停项目实施，对招标过程进行核查评估，如发现违规行为则重新组织招标。
